Display Settings

Display:none -> that the element will not be displayed

Display:block-> this is the default styling of HTML where each element takes up the whole width of the viewport

Display:inline-block-> it allows next element to take up the space that is left in the width

Display:inline-> it allows next element to take up the space that is left in the width but it does not allow to set height and width

Display:flex -> this enables flexbox
